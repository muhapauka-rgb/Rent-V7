# Проверка распознавания (OCR): краткая инструкция

## 1. Быстрая проверка, что OCR работает
1. Открой веб-интерфейс `http://localhost:8080`.
2. Выбери квартиру и текущий месяц.
3. Отправь фото в боте.
4. Проверь результат:
   - значение появилось в нужном поле;
   - если фото плохое — бот просит переслать;
   - если есть сильный скачок — бот пишет “передано администратору”.

## 2. Если распознано неверно
1. Исправь значение вручную в вебе.
2. Нажми сохранить.
3. Система запомнит факт исправления и использует это для обучения (через датасет).

## 3. Что означают цвета
- Оранжевым — ожидается фото по этому счётчику.
- Красным — требуется проверка/сомнение по распознаванию.

## 4. Проверка “умных повторов”
1. Отправь похожее фото с почти тем же значением.
2. Правило:
   - вода: допустимая разница до 1;
   - электро: допустимая разница до 5.
3. Если разница в пределах — значение перезапишется, без зацикливания.

## 5. Сообщение “подозрительный скачок”
Это значит:
1. Значение сильно отличается от прошлого месяца.
2. Нужна ручная проверка/правка администратором.
